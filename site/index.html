CloudPerch
MCP infrastructure for AI agents

A self-hosted gateway that gives AI agents the ability to browse the web, interact with platforms, and reach beyond the conversation window. — 30+ tools · 5 connectors · 1 endpoint

What it does

Large language models can reason but cannot act. They cannot open a webpage, read a comment section, search for a song, or post a reply. CloudPerch changes this by orchestrating multiple MCP servers behind a single HTTPS endpoint, giving any MCP-compatible client instant access to browser automation and platform-specific capabilities.

One domain. One TLS certificate. Many tools.

Capabilities

Browser Automation 8 tools
Navigate, screenshot, click, type, extract text, scroll, execute JavaScript. Headless Chromium via Playwright.

GitHub 70+ tools
